# Demo Scenarios

## 시나리오 1. 출장객

### 설정

- 항공편: `KE623`
- 현재 위치: 서울역
- 접근 방식: 자동차
- 이용자 유형: 출장객
- 선호 옵션: 가장 빠른 이동

### 시연 포인트

- AI가 권장 출발 시각과 보안검색 구역을 즉시 제안한다.
- 보안검색 혼잡 급증 이벤트를 넣으면 다른 구역으로 재배치한다.
- 심사위원 질문 대응:
  - 왜 단순 조회가 아닌가
  - 왜 사람이 직접 판단하는 것보다 빠른가

## 시나리오 2. 외국인 관광객

### 설정

- 항공편: `SQ607`
- 현재 위치: 홍대입구
- 접근 방식: 공항철도
- 이용자 유형: 외국인 관광객
- UI: English
- 선호 옵션: 면세점/식음 이용 선호

### 시연 포인트

- 영어 UI로 전환된다.
- 탑승 전 여유시간을 계산해 면세점/식음 추천을 제시한다.
- AI 어시스턴트에 `When should I leave now?`, `Do I have time for duty free?`를 입력해 상황형 답변을 보여준다.

## 시나리오 3. 교통약자/고령자

### 설정

- 항공편: `OZ501`
- 현재 위치: 송도
- 접근 방식: 자동차
- 이용자 유형: 교통약자/고령자
- 선호 옵션: 걷기 최소화

### 시연 포인트

- 보수적 버퍼 시간이 자동 반영된다.
- 접근성 우선 경로와 엘리베이터 허브 경유 동선이 선택된다.
- 위험도가 높아지면 조기 출발 권고가 더 강하게 표시된다.

## 추천 시연 순서

1. 대시보드에서 기존 정보 제공과의 차이를 설명한다.
2. 재조정 시뮬레이터에서 이벤트 전후 비교를 보여준다.
3. 단계별 화면으로 실제 사용 순간을 설명한다.
4. AI 어시스턴트에서 상황형 질문을 시연한다.
5. KPI 시뮬레이션에서 정량 기대효과를 마무리한다.
